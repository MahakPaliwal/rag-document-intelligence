# RAG Document Intelligence System

Ask natural language questions from any PDF using a free AI pipeline.

## Tech Stack
Python | LangChain | FAISS | HuggingFace | Groq API | PyMuPDF
## Live Demo
Try it here: [RAG Document Intelligence System](https://mahakpaliwal-rag-document-intelligence-app-sdyomo.streamlit.app/)

## How to Run
1. Open notebook in Google Colab
2. Add Groq API key in Colab Secrets as `GROQ_API_KEY`
3. Run all cells
4. Upload any PDF and start asking questions
